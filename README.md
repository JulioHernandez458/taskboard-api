# TaskBoard
Gestor de tareas personal tipo Trello. Permite organizar grupos, listas y tareas con estados: pendiente, en curso y hecho.

## Descripción general
Proyecto fullstack diseñado para practicar y demostrar habilidades modernas de desarrollo web.
Implementa autenticación JWT, operaciones CRUD, Spring Security y un frontend modular con Angular Material.

## Stack tecnológico
- Backend: Java 21 · Spring Boot 3 · Spring Security · JPA/Hibernate · PostgreSQL 
- Frontend: Angular 18 · Angular Material 
- Infraestructura: Docker · Docker Compose
